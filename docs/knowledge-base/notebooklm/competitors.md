## Camada de execução vs. "só software"

Ângulo de diferenciação (educativo): a maioria das plataformas de gamificação/engajamento foca **só no software** e tropeça na logística de prêmios físicos (claim "~85%" — **flag aprovação** para o número). A 4unik é a **camada de execução** plug-and-play: catálogo 5.000+, checkout embedded no app do parceiro, inventário, frete e tracking via API/webhook. Posicionar como complemento (não concorrente) das plataformas de software.

## Ângulos editoriais seguros

- Build vs. buy (engajamento + recompensas + logística).
- Checklist: API + gamificação + loja + rastreio.
- Reward infrastructure vs. swag vendor (mensagem repo).
- "Camada de execução" para plataformas de software (sem citar % sem aprovação).
