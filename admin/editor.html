Gestor da Landing Page Yoobe

v2.0

Este painel permite gerenciar todo o conteúdo da landing page atual sem mexer em código:

  • Hero & Topo — Título principal, subtítulo, badge de status, título da aba e links dos botões CTA (Agendar Demo / WhatsApp).
  • Serviços — Edite os 3 cards de serviços (Catálogo, Personalização, Logística).
  • API & Gamificação — Edite os painéis informativos de API e Gamificação.
  • Spoiler / Novidades — Gerencie o título do spoiler e os 3 cards de prévia (Dashboard, Loja, Produtos).
  • Clientes & Logos — Edite o título/descrição da seção, visualize as logos atuais, faça upload de novas e remova existentes.
  • Rodapé & Links — Edite o texto de copyright do rodapé.
  • Editor HTML — Acesso completo ao código-fonte para edições avançadas.
  • Pré-visualização — Veja a landing page em tempo real dentro do painel.
  • Backups — Cada edição gera um backup automático. Restaure versões anteriores a qualquer momento.
